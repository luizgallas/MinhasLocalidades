# MinhasLocalidades

### Aplicativo react-native de integração com a [API](https://servicodados.ibge.gov.br/api/docs/localidades?versao=1#api-Municipios-estadosUFMunicipiosGet) de localidades do IBGE. Feito para fins de estudos.
## Desenvolvido utilizando React Native, Typescript, Redux e Styled Components.
